# guoxk 的 Skills 集合

精选的 [Agent Skills](https://agentskills.io/home) 集合，涵盖 AI/Agent 框架、前端开发、Vue 生态以及开发者效率工具的最佳实践。

## 安装

```bash
pnpx skills add guoxk-me/skills --skill='*'
```

或全局安装：

```bash
pnpx skills add guoxk-me/skills --skill='*' -g
```

了解更多 CLI 用法，请查看 [skills](https://github.com/vercel-labs/skills)。

## 使用方法

### 安装指定 Skill

```bash
pnpx skills add guoxk-me/skills --skill='frontend-design'
```

### 安装所有 Skills

```bash
pnpx skills add guoxk-me/skills --skill='*'
```

### 在项目中使用 Skills

在 `AGENTS.md` 中引用 Skills：

```markdown
# Agents

## 前端开发
使用 [frontend-design](/skills/frontend-design) 进行 UI 实现。
使用 [vue](/skills/vue) 进行 Vue.js 开发。

## 测试
使用 [vitest](/skills/vitest) 进行单元测试。
```
